### Usage

For each color token, you can use the CSS variable or access it via the theme object in JS/TS:

```css
/* CSS/SCSS Variable */
var(--cds-color-primary)
var(--cds-color-onPrimary)
var(--cds-color-primaryContainer)
var(--cds-color-onPrimaryContainer)
/* ...and so on for each token */
```

```javascript
// JS/TS Theme Object
theme.colors.primary
theme.colors.onPrimary
theme.colors.primaryContainer
theme.colors.onPrimaryContainer
// ...and so on for each token
```

> **Accessibility Note:**  
> Always check color contrast, especially for text (`on...`) colors against their backgrounds, to ensure readability and accessibility compliance.
