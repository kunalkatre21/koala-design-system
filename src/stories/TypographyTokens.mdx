### Detailed Typography Tokens

#### display-large

<span style={{fontSize:'3.5625rem',lineHeight:'4rem',fontWeight:400,letterSpacing:'-0.02em',fontFamily:"'Roboto',sans-serif"}}>Sample Text</span>

**CSS Values:**
```
font-size: 3.5625rem;
line-height: 4rem;
font-weight: 400;
letter-spacing: -0.02em;
```
**Usage:** For large screen titles

**Code Snippets:**
```css
/* CSS/SCSS Variables */
font-size: var(--cds-typography-display-large-font-size);
line-height: var(--cds-typography-display-large-line-height);
font-weight: var(--cds-typography-display-large-font-weight);
letter-spacing: var(--cds-typography-display-large-letter-spacing);
```
```javascript
// JS/TS Theme Object
theme.typography.displayLarge
```
